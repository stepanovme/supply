# Реестр заявок

## Назначение

Реестр заявок — основной список всех заявок с быстрыми переходами к карточке, документам и связанным действиям.

## Что видно в реестре

- Номер и название заявки
- Проект
- Исполнитель
- Даты
- Текущий статус
- Прогресс по этапам

## Типовые действия

1. Открыть карточку заявки.
2. Прикрепить счёт.
3. Отправить на согласование.
4. Перейти к сопоставлению счёта.
5. Отслеживать исполнение.

## Рекомендации

> Используйте фильтры и поиск по ключевым полям, чтобы быстро находить активные заявки и узкие места в процессе.
